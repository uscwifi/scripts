# 本脚本在centos7下面测试没有问题
### lamp格组件的版本如下：
- httpd 2.4.6
- mysql 5.7。25
- php 7.0

### 使用方法：
- 进入目录下面，执行
```shell 
bash lamp.sh
```

- 没有多余的组件，能跑起来，没有任何优化，仅作为自己日后参考使用
